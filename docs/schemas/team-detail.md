**For team details:**
- Team id
- Team title
- Team city (optional)
- Team description (optional)
- Team color (optional)
- Team EESL ID (optional)
- Sponsor line ID (optional)
- Main sponsor ID (optional)
- Team logo URL (optional)
- Team logo icon URL (optional)
- Team logo web URL (optional)
- Sport id
