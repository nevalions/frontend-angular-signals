## What's on the page

- Search field for tournaments
- "Add Tournament" button
- List of tournament cards:
  - Trophy icon
  - Tournament name
  - Description (optional)
  - Click to go to tournament detail
- Items per page selector (10, 20, 50)
- Pagination controls
